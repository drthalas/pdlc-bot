# Decisions

- Mac mini remains the execution runtime.
- Railway will be the future dashboard/control center.
- Mac mini pushes events to Railway; Railway does not directly access Mac mini initially.
- Codex Runner is disabled by default.
- No automatic Codex execution without an explicit user click.
- No commit, push, or deploy without approval.
- Tester/QA Agent is mandatory in the future PDLC loop.
